# How to write a blog post

1. Write a blog post in markdown format and save it in the ```./md ``` directory.
2. Run ```python update.py```. This will update posts.json.

* Be sure to include the metadata in the beginning of each blog post .md file:

```
<!--
title: EXAMPLE_TITLE
date: 2025-08-10 (MUST BE YYYY-MM-DD FORMAT)
tags: [diffusion, ai, math]
-->

# TITLE

Here’s my note on score-based diffusion models...


[← Back to study index](index.html)
```
